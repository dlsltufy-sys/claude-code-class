## 종합소득세 신고계산기 완성

**4단계 구조로 구현:**

| 단계 | 내용 |
|------|------|
| 1단계 소득입력 | 근로·사업·금융·연금·기타소득, 소득공제 자동계산 |
| 2단계 소득공제 | 인적공제(기본+추가), 특별소득공제, 연금·노란우산 등 |
| 3단계 세액공제 | 자녀세액공제, 연금계좌, 특별세액공제(보험료·의료비·교육비·기부금) |
| 4단계 결과 | 단계별 계산과정 전체 표시, 지방소득세, 최종 납부/환급세액 |

**주요 기능:**
- 2024년 귀속 기준 8단계 누진세율 적용
- 실시간 우측 요약 패널 (입력 즉시 반영)
- 천단위 콤마 자동 포맷, 각 항목 툴팁 안내
- 금융소득 2,000만원 초과분 종합과세 처리

**실행 방법:**
```

```
또는 VS Code Live Server로 `index.html`을 열면 바로 사용 가능합니다.

## PDF 자동 입력 기능 추가 완료

헤더 우측에 **"PDF 자동 입력"** 버튼이 생겼어요.

### 지원 서류
| 서류 | 자동 입력 항목 |
|------|-------------|
| **근로소득 원천징수영수증** | 총급여, 국민연금, 건강+고용보험료, 기납부세액, 의료비, 교육비, 기부금 |
| **사업소득 원천징수영수증** | 총수입금액, 기납부세액 |
| **의료비 납입확인서** | 의료비 지출액 |
| **교육비 납입증명서** | 교육비 |
| **보험료 납입증명서** | 보장성 보험료 |
| **연금계좌 납입확인서** | 연금저축·IRP 납입액 |
| **국민연금/건강보험 납부확인서** | 각 보험료 |

### 사용법
1. "PDF 자동 입력" 버튼 클릭 → PDF 파일 선택
2. 분석 후 인식된 항목 목록 표시 → 원하는 항목만 체크
3. "선택 항목 입력 적용" 클릭 → 자동으로 해당 필드에 입력

> **주의**: 텍스트 기반 PDF(홈택스 발급본)만 지원. 스캔 이미지 PDF는 인식 불가
