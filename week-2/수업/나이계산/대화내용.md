# 나이 계산기 개발 대화 내용

## 1. 앱 생성 요청

**사용자:** 나이계산 폴더 안에 생년월일을 입력하면 나이를 보여주도록 만들어줘

**결과:**
- 생년월일 date input 추가
- 만 나이, 한국 나이, 태어난 지 며칠째, 다음 생일까지 남은 날짜 표시
- 오늘이 생일이면 축하 메시지 표시
- React + Tailwind CSS 기반 단일 HTML 파일로 구현

---

## 2. 윤석열 나이 추가 요청 (오해)

**사용자:** 윤석열 나이도 나오도록 해줘

**결과 (오해로 인한 잘못된 구현):**
- 윤석열 전 대통령의 생년월일(1960-12-18)을 기준으로 나이를 계산
- 사용자의 나이와 윤석열의 나이를 비교하는 "윤석열과 비교" 섹션 추가
- "윤석열보다 N살 많아요 / 어려요 / 동갑이에요" 메시지 표시

---

## 3. 의도 명확화 및 수정

**사용자:** 윤석열 나이를 넣어달라는게 아니고 윤석열 전 대통령이 바꾼 나이 계산법을 적용해 추가해달라는 말이야

**배경 지식:**
- 2023년 6월 28일 **만 나이 통일법** 시행
- 윤석열 전 대통령 재임 중 도입한 법령
- 모든 법령·공문서에서 만 나이를 공식 기준으로 사용하도록 통일

**한국의 3가지 나이 계산법:**

| 나이 종류 | 계산법 | 용도 |
|---|---|---|
| 세는 나이 | 올해 - 출생연도 + 1 | 일상 대화 (전통) |
| 연 나이 | 올해 - 출생연도 | 학교 입학, 군 입대 등 행정 |
| 만 나이 (✅ 공식) | 마지막 생일 기준 | 2023.6.28 이후 법정 표준 |

**최종 구현:**
- "윤석열과 비교" 섹션 제거
- 세는 나이 / 연 나이 / 만 나이 3가지를 나란히 표시
- 만 나이에 "공식" 배지 표시
- 만 나이 통일법 안내 문구 추가
