# 탕수육 레시피 제작 대화 기록

## 1. 탕수육 레시피 요청

**사용자:** 볶아 만들어도 바삭함을 오래 유지할 수 있는 탕수육 레시피 만들어줘

**Claude:**

바삭함의 핵심은 **튀김옷 배합**과 **2번 튀기기**라고 설명하며 다음 레시피를 제공.

### 핵심 비법
| 비법 | 설명 |
|------|------|
| 전분 황금 비율 | 감자전분 : 옥수수전분 : 박력분 = 3 : 1 : 1 |
| 얼음물 반죽 | 차가운 반죽이 수분을 폭발적으로 증발시킴 |
| 2번 튀기기 | 1차 160°C(속 익히기) → 2차 180°C(수분 제거) |

### 주요 재료 (3~4인분)
- 돼지고기 안심/등심 400g
- 튀김옷: 감자전분 6큰술, 옥수수전분 2큰술, 박력분 2큰술, 베이킹파우더, 얼음물, 달걀흰자
- 소스: 물, 설탕, 식초, 간장, 케첩, 물전분, 채소류

## 5. 재시도 요청

**사용자:** 그냥해줘 (×2)

**Claude:**

만료된 API 키 대신 **Pollinations.ai** (무료, 인증 불필요) 활용:

```python
import urllib.request, urllib.parse

prompt = "Korean Tang Su Yuk crispy golden fried pork sweet sour red sauce vegetables white plate food photography appetizing"
url = f"https://image.pollinations.ai/prompt/{urllib.parse.quote(prompt)}?width=800&height=450&nologo=true&seed=42"

with urllib.request.urlopen(url, timeout=60) as r:
    data = r.read()

with open("탕수육_썸네일.jpg", "wb") as f:
    f.write(data)
```

- `탕수육_썸네일.jpg` 생성 완료 (69,554 bytes)
- `탕수육_레시피.md` 상단에 이미지 삽입
